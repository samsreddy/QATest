# QATest
Sample QA Test for the public view

# Introduction
Test automation project that will be used to test TFL UI applications on the TFL site. This project comprises of sample test framework project to make sure
it reusable across multiple projects.

# Getting Started
To run this project locally follow the below steps
1. Install visual studio
2. Install specflow plugin in visual studio
3. Clone the repository and update nuget packages
